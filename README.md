# Universal Browser LLM Agent

Универсальный веб-агент с чатом, который работает прямо в DOM текущей страницы.
Проект поддерживает два формата:
- браузерное расширение (для личного использования в любом сайте);
- встраиваемый виджет (для интеграции в конкретный сайт/продукт).

## Для чего проект

- Помогать пользователю выполнять задачи на сайте через чат: найти товар, нажать кнопку, заполнить поле, собрать информацию.
- Снизить порог входа для сложных интерфейсов.
- Дать сайтам и командам поддержки готовый AI-слой без ручной интеграции действий под каждый экран.

## Для каких сегментов

- Частные пользователи: хотят автоматизировать рутину в браузере через расширение.
- Интернет-магазины и сервисы: хотят встроить помощника в сайт как виджет для своих клиентов.
- Команды поддержки и продукта: хотят сократить время выполнения типовых пользовательских сценариев.

## Когда что использовать 

- Расширение: если помощник нужен вам лично на разных сайтах и без доработки этих сайтов.
- Виджет: если вы владелец сайта/продукта и хотите дать чат-помощник всем посетителям внутри вашего интерфейса.

## Структура проекта

- `app/` — backend на FastAPI.
- `app/main.py` — HTTP API и статика виджета.
- `app/agent/` — оркестратор чата и правила выполнения действий.
- `app/llm/` — planner/finalizer и промпты LLM.
- `app/models.py` — Pydantic-модели запросов/ответов.
- `extension/` — файлы браузерного расширения (popup + content script).
- `widget/` — клиент виджета и iframe-чат для встраивания в сайт.
- `tests/` — тесты логики planner/safety/orchestrator.
- `requirements.txt` — зависимости Python.
- `universal-llm-agent-latest.crx` — последняя собранная версия расширения.
- `extension.pem` — ключ подписи для пересборки `.crx`.

## Как запустить

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m app.main
```

Backend по умолчанию: `http://127.0.0.1:8000`.

## Как пользоваться расширением


1. Возьми файл `universal-llm-agent-latest.crx`.
2. Открой страницу расширений в Браузере (напр. browser://tune/).
3. Перетащи `.crx` в окно браузера и подтверди установку.

После установки открой любой сайт, нажми иконку расширения и напиши задачу в чат.

## Как пользоваться виджетом

Добавь на сайт:

```html
<script>
  window.UniversalAgentConfig = {
    backendUrl: "https://your-agent-backend.example.com",
    siteId: "client_001",
    buttonLabel: "Помощник"
  };
</script>
<script async src="https://your-agent-backend.example.com/widget.js"></script>
```

После загрузки появится кнопка чата на странице. Пользователь общается с агентом внутри сайта, без расширения.

## Что умеет агент

- `extract_text` — искать и извлекать информацию со страницы.
- `click` — находить и нажимать элементы интерфейса.
- `type` — заполнять поля и формы.
- `press` — отправлять клавиши (например Enter).
- `wait_for` — ждать появления элемента.

Финальные необратимые действия (например оплата) запрашивают явное подтверждение.

## Ограничения

- На страницах `browser://`, `yandex://`, `chrome://` действия недоступны.
- Внутренности cross-origin iframe могут быть недоступны.
- CAPTCHA/2FA/антибот-защиты не обходятся.
